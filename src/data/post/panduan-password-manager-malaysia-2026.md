## 🔐 Apa Itu Password Manager?

Korang selalu dengar orang cakap "jangan guna password sama untuk semua akaun" kan? Tapi realitinya, manusia biasa memang susah nak ingat 20-30 password berbeza. Lagi-lagi kalau setiap satu kena ada huruf besar, nombor, simbol — memang pening.

Password manager adalah aplikasi yang jadi **pengurus kata laluan digital**. Ia simpan SEMUA password korang dalam satu tempat selamat, dan korang cuma perlu ingat SATU master password je. Lepas tu, ia auto-fill login masa korang nak masuk mana-mana website atau app.

Bayangkan macam dompet digital — tapi untuk password.

## 💡 Kenapa Korang Perlukan Password Manager

Sebelum kita pergi lebih dalam, faham dulu kenapa benda ni penting untuk orang Malaysia:

| Masalah | Tanpa Password Manager | Dengan Password Manager |
|---|---|---|
| Password senang diteka | Guna "password123" atau tarikh lahir | Auto-generate password 20 char rawak |
| Guna password sama | Satu akaun kena hack, semua habis | Setiap akaun password unik |
| Lupa password | Kena reset berkali-kali | Auto-fill, tak perlu ingat |
| Kena phish/scam | Senang tertipu | Auto-fill hanya di website sebenar |
| Kongsi password | WhatsApp atau tulis kat kertas | Kongsi secara selamat dalam app |

Statistik dari CyberSecurity Malaysia menunjukkan lebih 60% pengguna internet tempatan guna password sama untuk pelbagai akaun. Ini bahaya sebab kalau satu service kena data breach, semua akaun korang berisiko.

## 📱 Cara Setup Password Manager (5 Minit)

### Langkah 1: Download & Install

```
Android: Google Play Store → cari "Bitwarden"
iPhone: App Store → cari "Bitwarden"
Laptop: Firefox/Chrome store → add Bitwarden extension
```

### Langkah 2: Buat Akaun

- Buka app Bitwarden
- Klik "Create Account"
- Masukkan email dan **master password**
- Master password kena kuat —至少 12 aksara, campur huruf besar, kecil, nombor, simbol
- Tulis master password kat kertas dan simpan di tempat selamat (jangan digital!)

### Langkah 3: Install Browser Extension

- Chrome Web Store → "Bitwarden"
- Firefox Add-ons → "Bitwarden"
- Login guna master password
- Siap — bila korang log masuk mana-mana website, Bitwarden akan tanya nak simpan password atau tak

### Langkah 4: Mula Guna

1. Bila login website baru → Bitwarden tanya "Save this password?"
2. Klik save → lain kali auto-fill
3. Guna fitur **Password Generator** (ikon kocok) untuk create password kukuh
4. Tak perlu ingat lagi — Bitwarden uruskan semua

## 🚀 Tips Guna Password Manager Dengan Bijak

### 1. Guna Passphrase untuk Master Password

Master password adalah kunci kepada SEMUA password korang. Jadi kena betul-betul kuat.

**Cara buat passphrase:**
- Pilih 4-5 perkataan random: `kucing-terbang-pizza-kuning-buku!`
- Campur simbol dan nombor: `Kucing!Terbang42-Pizza`
- 20+ aksara, mudah diingat, susah diteka

Jangan guna master password yang ada kaitan dengan diri korang — tarikh lahir, nama haiwan peliharaan, alamat rumah.

### 2. Aktifkan 2FA

Double authentication = lapisan keselamatan tambahan. Walaupun ada orang tahu master password korang, dia still tak boleh masuk tanpa kod 2FA.

Guna authenticator app macam Google Authenticator atau Authy — elakkan SMS 2FA sebab SIM swap attack masih jadi masalah di Malaysia.

### 3. Audit Password Sedia Ada

Bitwarden ada fitur **Vault Health Report** yang scan semua password korang dan bagitahu mana yang:
- Lemah (weak)
- Diguna semula (reused)
- Terlibat dalam data breach (exposed)

Guna fitur ni untuk ganti password yang bermasalah satu persatu.

### 4. Jangan Simpan Master Password di Cloud

Ini kesilapan paling besar — orang simpan master password dalam Google Keep, Notes iPhone, atau WhatsApp. Kalau phone kena hack, semua password habis.

**Cara selamat:**
- Tulis kat kertas dan simpan dalam safe/laci berkunci
- Guna hardware wallet macam YubiKey untuk backup
- Kongsi dengan ahli keluarga dipercayai (dalam sampul surat bertutup)

### 5. Import dari Browser

Korang boleh import semua password yang dah disimpan dalam Chrome/Firefox terus ke Bitwarden:

```
Bitwarden App → Settings → Import Data → Pilih Chrome
```

Baru 5-10 minit siap. Lepas tu hapuskan password yang disimpan dalam browser — ia tak selamat sebab sesiapa yang guna laptop korang boleh tengok.

## 🛑 Perangkap Yang Kena Elak

### 1. "Password manager tak selamat — kalau kena hack habis semua"

Ini mitos paling besar. Password manager guna **zero-knowledge architecture** — maksudnya syarikat penyedia pun **tak boleh** access vault korang. Data disulitkan (AES-256) sebelum sampai ke server. Nak buka kena ada master password — yang hanya korang tahu.

Bandingkan dengan:
- Bahaya: Guna password sama untuk 20 akaun → 1 kena hack, 20 habis
- Selamat: Guna password manager → 20 password unik, cuma 1 master password perlu dijaga

### 2. "Free version mesti ada had"

Bitwarden free dah termasuk unlimited passwords, unlimited devices, dan semua fitur asas. Premium cuma tambah fitur mewah macam TOTP 2FA.

### 3. "Susah nak setup dan guna"

Setup ambil masa 10 minit lepas tu auto-pilot. Setiap kali login website baru, app tanya "save?" — klik je. Senang macam tu.
